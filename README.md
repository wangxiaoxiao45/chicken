### 如何运行
> - 下载项目   **git clone https://github.com/wxxfe/chicken.git**
> - 打开一个终端，进入到**chicken目录** 安装依赖包
>  `npm install`
>  如果失败，尝试用`yarn install`
> - 启动后台服务  `node server/index`
> - 启动前端开发服务 `npm run dev`
> 启动后会自动打开网址 http://localhost:8080/#/
